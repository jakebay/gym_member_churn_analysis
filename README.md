# gym_member_churn_analysis

## Overview
This project analyzes gym membership data to identify trends in member retention and churn. The goal was to understand when and why members are most likely to leave. 

## Tools Used 
- Google Sheets
- Data Cleaning
- Pivot Tables
- Data Visualization

## Data 
The dataset combines active and lost members into a single structured dataset for analysis. 

## Key Insights 
- Most members churn within the first six months
- Overall churn rate is approximately 48.5%
- The longest tenured members are in the 50+ age range.
- Couple's memberships are the least likely to churn.
